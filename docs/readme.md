stm32-proxy
===========

> {{description}}

What is it
----------

It proxies requests coming from an STM Nucleo Board to the net via USB.
Requests from the board must be encoded through a protobuf and sent over
the serial port.

Install
-------

Install it with

    npm install stm32-proxy

Usage
-----

~USAGE~
